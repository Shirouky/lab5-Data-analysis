# Лабораторная работа №5: Визуализация данных

## Описание
Визуализация данных из Excel-файла с товарами бисера: построение гистограмм, круговых диаграмм, линейных графиков и анализ экстремальных значений с использованием Matplotlib.

## Технологии
- **Python 3.7+**
- **Pandas** - обработка данных
- **NumPy** - числовые операции
- **Matplotlib** - визуализация данных

## Структура проекта
```
LW5.ipynb                 # основной скрипт визуализации
DataFrame beads.xlsx      # исходные данные
beads.json                # альтернативный источник данных
```

## Функциональность

### 1. Гистограмма цен товаров
- **Разбиение на 10 групп** для анализа распределения цен
- **Визуализация концентрации** ценовых диапазонов
- **Выводы**: Цены в основном сосредоточены вокруг 180-200 рублей

### 2. Круговая диаграмма категорий
- **Топ-5 категорий** с наибольшим количеством товаров
- **Процентное соотношение** между категориями
- **Выводы**: Скидка в основном небольшая (4 или 6 процентов)

### 3. Линейные графики цен по категориям
- **5 подграфиков** для разных категорий товаров
- **Единая ось Y** для сравнения ценовых диапазонов
- **Выводы**: 
  - Голубой цвет продается стабильно по низкой цене
  - У красного цвета есть выброс
  - Розовый, оранжевый и серый имеют устойчивую цену около 200 рублей

### 4. Анализ максимальной цены
- **Выделение окрестности** ±100 объектов вокруг максимальной цены
- **Аннотация** с указанием значения максимальной цены
- **Выводы**: Максимальная цена является выбросом, в окрестности цены значительно ниже
